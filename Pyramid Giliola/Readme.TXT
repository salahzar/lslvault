Questo programma è da usarsi in OpenSIm per fare giochini.

Per realizzarlo usate phoenix viewer e importate il file xml Piramide Giliola.xml.

Nei vari pezzi (biglie linkate) mettete lo script "Elemento Piramide.lsl"

Nella piramidina mettete lo script "Pivot Piramide.lsl"

IMPORTANTE:
* selezionate tutti gli oggetti e fate RESET Script in Selection / REIMPOSTA oggetti in selezione
* cliccate la piramidina e scegliete REGISTRA per registrare la posizione corrente
* l'owner può selezionare REGISTRA, DISFA, RIMETTI A POSTO, 
* 

I suoni li potete mettere o uploadare i vostri.
